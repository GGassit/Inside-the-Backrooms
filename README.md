下载后如何编译？

一：
  首先你需要Dump游戏的所有c# "DLL"文件,然后引用在源码内（其中需要什么自己查看）。
  
  最后你需要去下载支持的注入器"MelonLoader",在github上搜索即可下载。
  
二：
  你需要下载目标框架版本：.NET Framework4.7.2，可支持C#的Visual Studio或者更多。
三：
  关于我们 QQ群：473740107。
